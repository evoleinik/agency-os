# DEV TASK
GOAL: Add a user profile page with avatar upload

## Objective
Create a new user profile page that allows users to view and edit their profile information, including uploading a profile avatar.

## Requirements
- Display current user information (name, email, joined date)
- Allow editing name and bio
- Support avatar upload (max 2MB, jpg/png only)
- Show success/error messages for operations
- Mobile responsive design

## Acceptance Criteria
- Profile page accessible at /profile
- Avatar upload works and shows preview
- Form validation prevents invalid inputs
- Changes persist after page refresh
- No console errors
